```text
[학생] 웹페이지에서 인적사항(학년/반/번호/이름) 및 종목 선택 후 제출

       ├── ❌ 명단과 안 맞을 때: "등록된 학생 명단에 일치하는 인적사항이 없습니다" 경고 반환!
       └── ⭕ 명단 일치 시: 본인 확인 완료 ➔ [설문응답] 탭에 100% 안전 덮어쓰기/저장!
```

## 📋 구글 시트에 학생 명단 넣는 방법 (단 10초 완료!)

1. 선생님의 비공개 구글 스프레드시트를 엽니다.
2. 아래 탭에서 **새 시트 추가(+)**를 누르고 이름을 **`학생명단`**으로 입력합니다.
3. 선생님의 기존 엑셀 명단 파일에서 학년, 반, 번호, 이름 열을 마우스로 복사(Ctrl+C)합니다.
4. **`학생명단`** 탭의 A1 셀에 붙여넣기(Ctrl+V) 합니다:
   - **A열**: 학년 (예: `1학년`)
   - **B열**: 반 (예: `1반`)
   - **C열**: 번호 (예: `1번`)
   - **D열**: 이름 (예: `홍길동`)

> 💡 **참고**: `학생명단` 탭에 데이터를 넣어두지 않으면 명단 검증 단계가 생략되고 모든 제출을 수락합니다.

## 🛠️ 앱스 스크립트 설정 (`code.gs`)

1. [code.gs](file:///c:/Users/USER/Desktop/work/student-registration/code.gs)의 전체 코드를 구글 시트의 **[확장 기능] ➔ [Apps Script]**에 복사합니다.
2. **[배포] ➔ [새 배포]**를 누르고 아래와 같이 설정합니다:

   - 실행 주체: **나 (Me)**
   - 액세스 권한이 있는 사용자: **모든 사용자 (Anyone)**
3. 생성된 **웹 앱 URL**을 웹사이트의 관리자 모드에 저장하면 연동 완료됩니다!
